Installation Guide

System Requirements

  • iOS Version: iOS 15.0 or later
  • Devices: iPhone and iPad
  • Storage: 50 MB available space
  • iCloud Account: Required for CloudKit sync feature
  • Internet Connection: For real-time synchronization

Installation Steps

  1. Download from App Store - Search for "Wellbeing Log"
  2. Grant Permissions - Allow file access for CSV import/export
  3. Sign into iCloud - Required for CloudKit sync functionality
  4. Enable iCloud for App - Go to Settings > [Your Name] > iCloud > Apps using iCloud
  5. Add Your First Students - Use the main menu to add student records
‚úÖ Ready to Use! Once installed and configured, you can immediately start tracking student wellbeing and attendance.

CloudKit Setup

What is CloudKit?

CloudKit is Apple's professional infrastructure for real-time data synchronization between devices. It provides secure, reliable, and automatic sync capabilities without requiring external servers.

Setup Requirements

  • iCloud Account: Must be signed in on all devices
  • Same iCloud Account: All devices must use the same iCloud account
  • Internet Connection: Required for initial sync and updates
  • App Permissions: Enable iCloud for Wellbeing Log in Settings

Setup Steps

  1. Sign into iCloud on all devices you want to sync
  2. Enable iCloud for Wellbeing Log in Settings > [Your Name] > iCloud > Apps using iCloud
  3. Grant CloudKit permissions when prompted
  4. Access "CloudKit Room Sync" from the main menu
  5. Test sync functionality by making changes on one device and checking another
‚ö†Ô∏è Important: CloudKit sync requires an iCloud account. This is a free Apple service that provides secure, professional-grade synchronization.

Benefits of CloudKit

  • Real-time Updates: Changes appear on all devices within 30 seconds
  • Automatic Conflict Resolution: CloudKit handles simultaneous changes intelligently
  • Offline Support: Changes sync when connection is restored
  • Secure Infrastructure: All data stays within Apple's secure ecosystem
  • No External Servers: No third-party services or data transmission

Usage Instructions

Basic Operations

1. Adding Students

  1. From the main menu, select "Add Students"
  2. Enter student information (name, year, teacher)
  3. Save the student record
  4. Students are now available for room tracking

2. Tracking Student Movement

  1. Student In: Select a room, choose a student, and record their entry with a reason
  2. Student Out: Find students currently in a room and record their exit with an outcome
  3. Head Count: View current student counts and lists for each room

3. Monitoring Room Activity

  • Wellbeing Room: Track students seeking emotional support
  • Diverse Learners Room: Monitor students with learning support needs
  • Lunch Room: Manage lunchtime attendance

4. Generating Reports

  1. Access "Statistics" from the main menu
  2. View comprehensive reports on student movements
  3. Export data via CSV for external analysis
  4. Monitor room usage patterns and trends

CloudKit Sync Usage

  1. Ensure all devices are signed into the same iCloud account
  2. Access "CloudKit Room Sync" from the main menu
  3. Monitor sync status and connection
  4. Changes automatically appear on all connected devices
  5. Use "Manual Sync Now" button for immediate updates

Troubleshooting

Common Issues & Solutions

CloudKit Sync Issues

  • Problem: "iCloud Disconnected" error
  • Solution: Check iCloud sign-in and internet connection in Settings
  • Problem: "Sync Failed" error
  • Solution: Verify CloudKit permissions and iCloud storage availability
  • Problem: No updates appearing on other devices
  • Solution: Ensure all devices are signed into the same iCloud account

App Functionality Issues

  • Problem: App crashes or freezes
  • Solution: Update to latest iOS version and restart the app
  • Problem: Permission errors
  • Solution: Grant necessary permissions in Settings > Privacy & Security
  • Problem: Data not saving
  • Solution: Check device storage and file access permissions

Performance Issues

  • Problem: Slow app performance
  • Solution: Close other apps and restart Wellbeing Log
  • Problem: Sync delays
  • Solution: Check internet connection and iCloud status
üí° Pro Tip: If you experience persistent issues, try signing out of iCloud and signing back in, then restart the app.
